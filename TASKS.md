- [x] **内容**: AI 分析 API 呼び出しを **AI ボタンがクリックされたときだけ** 実行するロジックを実装。

- **目的**: ユーザー体験を損なわず、クリックあたりの収益を向上させる。

- **内容**: Cloudflare にアカウント作成 → ドメイン追加 → DNS を Cloudflare に切り替える。
- **所要時間**: 約 30 分
- **目的**: タイル配信の CDN キャッシュ化と将来のスケールアップ基盤構築。

## 6️⃣ cron スクリプトテンプレート作成

- **内容**: `scripts/update_tileserver.sh`（Docker イメージ自動更新） と `scripts/update_osm.sh`（OSM データ定期取得） を作成し、実行権限を付与。
- **所要時間**: 約 20 分
- **目的**: タイルサーバーとデータの自動メンテナンス。

## 7️⃣ モニタリング・ログ設計

- **内容**: Google Analytics に **`ai_click`** イベントを設定し、VPS の CPU/帯域・広告・AI の利用状況をログで取得する方針をまとめる。
- **所要時間**: 約 15 分
- **目的**: 収益・コストの可視化とスケール判断材料の確保。

## 8️⃣ README 更新

- **内容**: `README.md` に **「次にやること」** セクションを追加し、上記タスクと所要時間を記載。
